# 后期制作

## 推荐剪辑工具

| 工具 | 平台 | 适合场景 |
|------|------|----------|
| **剪映** | 手机/电脑 | 快速拼接，字幕自动生成，免费 |
| **CapCut** | 手机/电脑 | 剪映国际版，功能相同 |
| **Premiere Pro** | 电脑 | 专业剪辑，精细控制 |
| **DaVinci Resolve** | 电脑 | 免费专业剪辑，调色强大 |

## 拼接步骤（以剪映为例）

```
步骤 1：新建项目，画布比例设为 1:1
步骤 2：导入所有视频片段（甲骨文→金文.mp4、金文→小篆.mp4、小篆→楷书.mp4）
步骤 3：按顺序排列在时间线上
步骤 4：片段间添加"叠化"或"淡入淡出"转场（0.3-0.5 秒）
步骤 5：在每个片段上添加字幕（见下方模板）
步骤 6：最后一个片段后加 1 秒静止画面（楷书定格）
步骤 7：导出 MP4（1080×1080，码率 8-10 Mbps）
```

## 字幕设计

### 字幕内容

```
片段 1（甲骨文→金文）：
  字幕：商朝 甲骨文 — 手抓耳朵

片段 2（金文→小篆）：
  字幕：西周 金文 — 字形规整化

片段 3（小篆→楷书）：
  字幕：秦朝 小篆 — 线条圆润统一

结尾定格：
  字幕：现代 楷书 — "取" qǔ = take, get
```

### 字幕样式

- 字体：思源黑体 (Noto Sans SC) 或微软雅黑
- 字号：画面高度的 5-8%
- 颜色：白色字 + 黑色描边/阴影
- 位置：画面底部 15% 区域

## 旁白脚本（可选）

每段一句话，总时长控制在 10 秒内：

```
"商朝士兵打完仗，用手割下敌人的耳朵。
这个'取'字，就是一只手抓着一只耳朵。
后来字形越来越规整，变成了今天我们写的'取'。
取，qǔ，意思是 to take, to get。"
```

## 音效与配乐

- **背景音乐**：古风轻音乐（古琴、箫），音量调低
- **音效**（可选）：
  - 甲骨文阶段：刻字声（刀刻骨头）
  - 金文阶段：铸造声（青铜器浇铸）
  - 小篆/楷书：毛笔书写声
- **音效来源**：[爱给网 aigei.com](https://www.aigei.com) 或 [freesound.org](https://freesound.org) 搜索"calligraphy sound effect"

## 导出设置

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| 分辨率 | 1080×1080 | 正方形，适合手机端 |
| 帧率 | 30fps | 标准帧率 |
| 编码 | H.264 | 兼容性最好 |
| 码率 | 8-10 Mbps | 画质和大小的平衡 |
| 格式 | MP4 | 通用格式 |

## GIF 制作

### 方法 1：在线转换

1. 用剪映导出 MP4
2. 打开 [ezgif.com/mp4-to-gif](https://ezgif.com/mp4-to-gif)
3. 上传 MP4
4. 设置尺寸 480×480，帧率 15fps
5. 下载 GIF

### 方法 2：FFmpeg 命令行

```bash
ffmpeg -i input.mp4 -vf "fps=15,scale=480:-1:flags=lanczos,split[s0][s1];[s0]palettegen[p];[s1][p]paletteuse" -loop 0 output.gif
```
