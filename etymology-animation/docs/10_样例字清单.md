# 样例字清单

## P0 — 象形字（简单直观，建议先做）

| 字 | 甲骨文像什么 | 故事脚本 | 建议段数 |
|----|-------------|----------|----------|
| 日 | 圆圈中间一点，太阳 | 圆太阳 → 逐渐变方 → 楷书"日" | 1 段 |
| 月 | 弯弯的月牙 | 月牙 → 逐渐变方 → 楷书"月" | 1 段 |
| 山 | 三座尖峰 | 三座山峰 → 逐渐变平 → 楷书"山" | 1 段 |
| 水 | 弯曲河流，两边水花 | 河流 → 曲线变直 → 楷书"水" | 1 段 |
| 火 | 火苗燃烧 | 火苗 → 逐渐简化 → 楷书"火" | 1 段 |
| 人 | 侧身弯腰的人 | 人弯腰 → 逐渐简化 → 楷书"人" | 1 段 |
| 木 | 一棵树，有枝有根 | 树 → 逐渐简化 → 楷书"木" | 1 段 |
| 目 | 竖着的眼睛 | 眼睛 → 逐渐变方 → 楷书"目" | 1 段 |
| 耳 | 一只耳朵 | 耳朵 → 逐渐简化 → 楷书"耳" | 1 段 |

## P1 — 会意字（故事性强，最推荐）

| 字 | 部件组合 | 故事脚本 | 建议段数 |
|----|---------|----------|----------|
| 取 | 手 + 耳 | 手抓耳朵 → 割耳 → "取" | 3 段 |
| 采 | 爪 + 木 | 手在树上摘果子 → "采" | 2 段 |
| 休 | 人 + 木 | 人靠树旁休息 → "休" | 2 段 |
| 明 | 日 + 月 | 日月并列 → 光明 → "明" | 1 段 |
| 森 | 三木 | 三棵树 → 森林 → "森" | 1 段 |
| 看 | 手 + 目 | 手搭凉棚看远方 → "看" | 1 段 |
| 男 | 田 + 力 | 田里干活的人 → "男" | 1 段 |
| 安 | 宀 + 女 | 女子在屋子里 → 安全 → "安" | 1 段 |
| 家 | 宀 + 豕 | 屋子里有猪 → 家 → "家" | 1 段 |
| 信 | 人 + 言 | 人说的话 → 诚信 → "信" | 1 段 |
