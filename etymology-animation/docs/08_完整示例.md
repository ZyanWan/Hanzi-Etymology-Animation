# 完整工作示例："取"字

从零到成品的完整流程，可直接照做。

## 第 1 步：研究

```
【字】：取
【读音】：qǔ
【本义】：割取敌人的耳朵（古代战争中割耳计功）
【字形来源】：
  - 甲骨文：左耳右手，像手抓耳朵
  - 金文：形状规整，耳和手更像符号
  - 小篆：圆润统一，"耳"在左"又"在右
  - 楷书：今天的"取"
【故事脚本】：士兵用手抓敌人耳朵 → 演变为"取"
```

## 第 2 步：素材

```
1. 打开 zdic.net，搜索"取"
2. 点击"字源"标签
3. 分别保存甲骨文、金文、小篆的字形图片
4. 用图片编辑工具统一处理为 1024×1024 白底黑字 PNG
5. 保存到素材库/取/目录下：
   - 01_甲骨文.png
   - 02_金文.png
   - 03_小篆.png
   - 04_楷书.png（用现代楷书字体制作）
```

## 第 3 步：AI 视频生成

### 片段 1：甲骨文 → 金文

```
工具：可灵，首尾帧模式
起始帧：01_甲骨文.png
结束帧：02_金文.png
Prompt：白底黑字的商朝甲骨文字形，平滑变形过渡为西周金文字形，
       保持字形完整不变形，笔画自然过渡，书法风格，画面稳定，2秒
参数：高质量，1:1，运动幅度 40%，创意度 0.6
→ 下载 取_片段1.mp4
```

### 片段 2：金文 → 小篆

```
工具：可灵，首尾帧模式
起始帧：02_金文.png
结束帧：03_小篆.png
Prompt：白底黑字的西周金文字形，平滑变形过渡为秦朝小篆字形，
       线条从粗犷变为均匀圆润，保持字形完整不变形，书法风格，画面稳定，2秒
参数：高质量，1:1，运动幅度 40%，创意度 0.6
→ 下载 取_片段2.mp4
```

### 片段 3：小篆 → 楷书

```
工具：可灵，首尾帧模式
起始帧：03_小篆.png
结束帧：04_楷书.png
Prompt：白底黑字的秦朝小篆字形，平滑变形过渡为现代楷书字形，
       线条从圆润变为方折，保持字形完整不变形，楷书风格，画面稳定，2秒
参数：高质量，1:1，运动幅度 40%，创意度 0.6
→ 下载 取_片段3.mp4
```

## 第 4 步：后期组装

```
1. 打开剪映，新建 1:1 项目
2. 导入 取_片段1.mp4、取_片段2.mp4、取_片段3.mp4
3. 按顺序排列在时间线
4. 片段间加"叠化"转场（0.3 秒）
5. 添加字幕：
   - 片段 1："商朝 甲骨文 — 手抓耳朵"
   - 片段 2："西周 金文 — 字形规整"
   - 片段 3："秦朝 小篆 → 现代 楷书"
6. 最后加 1 秒楷书定格，字幕："取 qǔ = take, get"
7. 导出 MP4（1080×1080，H.264，10Mbps）
8. 用 ezgif.com 转换为 GIF（480×480，15fps）
```

## 第 5 步：嵌入

```
1. 将 取.gif 和 取.mp4 放入 courses/hsk5/L2_.../assets/etymology/
2. 在 HTML 的"取"词汇卡片背面添加"字源演变"按钮
3. 添加 showEtymology('取') 的 JS 逻辑
4. 在 etymologyData 中注册"取"的数据
```
