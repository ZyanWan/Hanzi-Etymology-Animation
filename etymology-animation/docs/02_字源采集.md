## 🎯 工具链概述

### 第一阶段：字源数据采集

**工具**：`scraper/` 模块

**功能**：
- 从汉典网站(zdic.net)自动采集汉字字源数据
- 下载完整演变链图片：甲骨文→金文→小篆→隶书→楷书
- 获取标准字体：楷书(5地区)、康熙字典、说文解字
- 生成JSON元数据，包含Unicode、拼音、字义等

**数据来源**：汉典网站（CC0协议，可商用）

**输出**：
- 高质量SVG矢量图片（可无损缩放）
- 结构化JSON元数据

---

### 第二阶段：图片预处理

**工具**：需配合图像处理库（如Pillow）

**功能**：
- SVG转PNG（推荐尺寸：1024×1024像素）
- 统一背景（白色背景，黑色字形）
- 图片命名规范化

**建议命令**：
```bash
# 批量转换SVG到PNG（需要安装ImageMagick或类似工具）
magick mogrify -format png -size 1024x1024 -background white -fill black data/raw/*/*.svg
```

---

### 第三阶段：AI视频生成

**工具**：阿里云百炼 Wan2.7、可灵AI等视频生成平台

**输入**：
- 起始帧：甲骨文或金文图片
- 结束帧：现代楷书图片
- Prompt：描述字形演变过程

**输出**：
- 字形演变动画视频
- 建议时长：6-8秒
- 格式：MP4/GIF

## 🚀 快速开始

### 方式1：命令行使用

```bash
# 采集单个汉字
python -m scraper.main 取

# 采集多个汉字
python -m scraper.main 日 月 山 水

# 采集P0-P2优先级样例（24个汉字）
python -m scraper.main -s

# 只采集数据，不下载图片
python -m scraper.main -s --no-images

# 保存为批量数据文件
python -m scraper.main -s --batch
```

### 方式2：Python API

```python
from scraper import EtymologyCollector

# 初始化采集器
collector = EtymologyCollector()

# 采集单个汉字
result = collector.collect_single("取", download_images=True)
print(f"成功下载 {len(result['images'])} 张图片")

# 批量采集
stats = collector.collect_batch(["日", "月", "山", "水"], download_images=True)
print(f"成功: {stats['success']}/{stats['total']}")

# 关闭资源
collector.close()
```

### 方式3：读取汉字列表文件

```bash
# 创建汉字列表文件
echo "取" > chars.txt
echo "采" >> chars.txt
echo "休" >> chars.txt

# 从文件读取采集
python -m scraper.main -f chars.txt
```

## 🔧 PlayWright集成指南

### 为什么需要PlayWright？

虽然`scraper`模块使用requests进行网页爬取，但**Playwright可以用于**：

1. **验证和测试**：确认采集的图片在浏览器中显示正确
2. **动态内容处理**：处理JavaScript渲染的内容
3. **截图对比**：生成演变动画的预览截图
4. **自动化工作流**：串联整个制作流程

### PlayWright工作流

详细内容请查看 [playwright_workflow.md](playwright_workflow.md)

### 常用PlayWright命令

```bash
# 安装PlayWright（如果尚未安装）
npx playwright install chromium

# 打开网页并截图
npx playwright screenshot https://www.zdic.net/hans/取 --full-page screenshot.png

# 获取页面所有图片URL
npx playwright evaluate https://www.zdic.net/hans/取 "document.querySelectorAll('img').length"

# 截取特定区域
npx playwright screenshot https://www.zdic.net/hans/取 --selector ".entry-gallery" gallery.png
```

### Python Playwright集成

```python
from playwright.sync_api import sync_playwright

# 示例：验证采集的图片URL
with sync_playwright() as p:
    browser = p.chromium.launch()
    page = browser.new_page()

    # 打开汉典页面
    page.goto("https://www.zdic.net/hans/取")

    # 获取所有字源图片
    images = page.query_selector_all('.entry-gallery img')

    for img in images:
        src = img.get_attribute('src')
        print(f"图片URL: {src}")

    browser.close()
```

## 🤖 AI自动化建议

### 完整AI自动化流程

```python
# AI自动化脚本示例：complete_pipeline.py

import subprocess
import json
from pathlib import Path

def ai_automation_pipeline():
    """
    AI全自动化字形演变动画制作流程
    """

    # Step 1: 采集字源数据
    print("Step 1: 采集字源数据...")
    subprocess.run(["python", "-m", "scraper.main", "取"])

    # Step 2: 读取元数据
    print("Step 2: 读取元数据...")
    with open("data/metadata/char_取_53D6.json", "r", encoding="utf-8") as f:
        data = json.load(f)

    # Step 3: 选择关键演变阶段
    print("Step 3: 选择关键阶段...")
    key_stages = select_key_stages(data)

    # Step 4: 生成Prompt
    print("Step 4: 生成Prompt...")
    prompt = generate_prompt(data)

    # Step 5: 调用AI视频生成API
    print("Step 5: 调用AI视频生成...")
    # 这里是占位，实际需要接入视频生成API
    video_url = call_video_api(key_stages, prompt)

    return video_url

def select_key_stages(data):
    """选择关键演变阶段"""
    evolution = data.get("evolution_images", {})

    # 建议选择顺序
    stages = [
        ("jiaguwen", 0),      # 甲骨文
        ("jinwen", 0),         # 金文
        ("xiaozhuan", 0),      # 小篆
        ("lishu", 0),          # 隶书
        ("kaishu_cn", -1),     # 楷书
    ]

    selected = []
    for stage_type, idx in stages:
        if stage_type in evolution and evolution[stage_type]:
            selected.append(evolution[stage_type][idx])

    return selected

def generate_prompt(data):
    """生成AI视频生成Prompt"""
    char = data.get("character", "")
    analysis = data.get("basic_info", {}).get("analysis", "")

    prompt = f"""
    汉字'{char}'的字形演变动画。

    字形分析：{analysis}

    要求：
    - 展现从甲骨文到楷书的演变过程
    - 保持字形结构的一致性
    - 笔画逐渐规整化
    - 时长6-8秒
    """

    return prompt.strip()
```

## ⚠️ 注意事项

### 1. 请求频率控制
- 系统已内置延迟（0.5秒/请求）
- 建议批量采集时分批进行
- 每次采集间隔至少30秒

### 2. 图片使用规范
- 图片来源于汉典网站（CC0协议）
- 建议在作品中标注图片来源
- SVG格式可无损缩放

### 3. 网络稳定性
- 采集过程中保持网络连接
- 失败会自动重试3次
- 建议定期备份采集数据

### 4. 存储空间
- 每个汉字约40-50个SVG文件
- 单个文件约10-50KB
- 1000个汉字约需500MB
