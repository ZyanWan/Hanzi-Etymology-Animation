# 嵌入学习页面

## 输出格式选择

| 格式 | 优点 | 缺点 | 建议用途 |
|------|------|------|----------|
| **GIF** | 自动循环播放，无需点击 | 文件大，无声音 | 简单展示 |
| **MP4** | 画质好，有声音，文件小 | 需要点击播放 | 完整版 |
| **WebM** | 网页原生支持 | 兼容性不如 MP4 | 高质量替代 |

**建议**：学习页面内嵌 GIF（自动播放），提供按钮链接到 MP4（有旁白）。

## 字源弹窗 JS

```javascript
const etymologyData = {
  '取': {
    story: '商朝士兵打完仗，用手割下敌人的耳朵。"取"就是一只手抓着一只耳朵。',
    gif: 'assets/etymology/取.gif',
    stages: ['甲骨文', '金文', '小篆', '楷书']
  },
  '采': {
    story: '一只手在树上摘果子。"采"就是手（爪）在树（木）上。',
    gif: 'assets/etymology/采.gif',
    stages: ['甲骨文', '金文', '楷书']
  },
  '休': {
    story: '一个人靠在树旁休息。"休"就是人（亻）靠着树（木）。',
    gif: 'assets/etymology/休.gif',
    stages: ['甲骨文', '金文', '楷书']
  }
};

function showEtymology(char) {
  const data = etymologyData[char];
  if (!data) return;

  const modal = document.createElement('div');
  modal.className = 'etymology-modal active';
  modal.innerHTML = `
    <div class="etymology-content">
      <span class="etymology-close" onclick="this.parentElement.parentElement.remove()">×</span>
      <h3 style="font-family: var(--serif); margin-bottom: 12px;">${char} — 字源演变</h3>
      <img src="${data.gif}" alt="${char}字形演变" style="margin-bottom: 12px;">
      <p style="font-size: 14px; color: var(--ink-light); line-height: 1.6;">${data.story}</p>
      <p style="font-size: 12px; color: var(--ink-muted); margin-top: 8px;">
        演变阶段：${data.stages.join(' → ')}
      </p>
    </div>
  `;
  modal.addEventListener('click', (e) => {
    if (e.target === modal) modal.remove();
  });
  document.body.appendChild(modal);
}
```
