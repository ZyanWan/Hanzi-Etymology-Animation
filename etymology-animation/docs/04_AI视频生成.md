# AI 视频生成

## 工具选择与对比

| 工具 | 网址 | 推荐理由 | 劣势 | 建议用途 |
|------|------|----------|------|----------|
| **可灵 (Kling)** | klingai.kuaishou.com | 国产，中文元素理解好，有"首尾帧"模式 | 偶尔笔画变形 | 备选工具 |
| **Sora** | sora.com | 效果精细，运动自然 | 需要 ChatG Plus | 高质量成品 |
| **Runway Gen-4** | runwayml.com | 运动控制精确 | 对中文元素不友好 | 英文 Prompt 时用 |
| **Vidu** | vidu.cn | 有"角色一致性"功能 | 字形变形效果待验证 | 可以尝试 |
| **海螺 AI** | hailuoai.video | 免费额度多 | 质量一般 | 快速原型 |

## 核心策略：分段生成，手动拼接

不要让 AI 一次性生成完整的甲骨文→楷书动画。分段效果更好：

```
视频片段 1：甲骨文 → 金文（1.5-2 秒）
视频片段 2：金文 → 小篆（1.5-2 秒）
视频片段 3：小篆 → 楷书（1.5-2 秒）
总时长：4.5-6 秒（加上转场和定格约 6-8 秒）
```

每段单独检查，不满意就重做，最后用剪映拼接。

## 可灵 (Kling) 详细操作

### 注册与登录

1. 打开 klingai.kuaishou.com
2. 用手机号或快手账号登录
3. 新用户有免费额度

### 生成单段动画

```
步骤 1：点击"AI 视频" → 选择"图生视频"
步骤 2：上传起始图片（如 01_甲骨文.png）
步骤 3：如果支持"首尾帧"模式 → 同时上传结束帧（如 02_金文.png）
步骤 4：填写 Prompt（见 04_Prompt集.md）
步骤 5：设置参数（见下方参数表）
步骤 6：生成，等待 1-3 分钟
步骤 7：预览，检查字形是否准确
步骤 8：不满意则调整 Prompt 或参数重新生成
步骤 9：满意后下载 MP4
```

### 首尾帧模式（强烈推荐）

同时上传起始图和结束图，AI 只负责生成中间过渡。字形准确性由你提供的图片保证，AI 只做动画。效果比纯文字 Prompt 可控得多。

### 参数设置

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| 模式 | 高质量 / 专业模式 | 画质优先 |
| 时长 | 2 秒 | 字形变形不需要太长 |
| 画面比例 | 1:1 | 正方形，和素材一致 |
| 运动幅度 | 30-50% | 太高会过度扭曲 |
| 创意度 | 0.5-0.7 | 太高会偏离原始字形 |

## Sora 操作流程

```
步骤 1：打开 sora.com → 用 ChatGPT Plus 账号登录
步骤 2：点击"Create video"
步骤 3：上传起始图片（如果支持图生视频）
  或者直接用文字 Prompt 生成
步骤 4：填写英文 Prompt
步骤 5：设置参数：
  - Duration: 2s
  - Resolution: 1080p
  - Aspect ratio: 1:1
步骤 6：生成 → 预览 → 下载
```

## Runway Gen-4 操作流程

```
步骤 1：打开 runwayml.com → 登录
步骤 2：选择"Generate Video" → "Image to Video"
步骤 3：上传起始图片
步骤 4：填写英文 Prompt
步骤 5：设置参数：
  - Duration: 2s
  - Resolution: 1280×720 或 1:1
  - Motion amount: 3-5（1-10 刻度）
  - Seed: 固定一个数字（方便复现）
步骤 6：生成 → 预览 → 下载
```

## 运动幅度 / 创意度调节

| 参数 | 太低 | 适中 | 太高 |
|------|------|------|------|
| 运动幅度 | 几乎不动 | 平滑变形 | 过度扭曲 |
| 创意度 | 过于保守，过渡生硬 | 保持字形同时有变化 | AI 自由发挥，字形走样 |

如果字形变形严重 → 降低这两个值。如果过渡太生硬 → 适当提高运动幅度。
