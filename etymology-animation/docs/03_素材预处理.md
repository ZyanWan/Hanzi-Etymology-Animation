# 素材准备

## 字形图片来源

| 来源 | 网址 | 用法 | 特点 |
|------|------|------|------|
| 汉典 | zdic.net | 搜字 → 字源 → 右键保存图片 | 最权威，有甲骨文/金文/小篆对照 |
| 象形字典 | vividict.com | 搜字 → 看演变图 | 有详细字源故事，适合写脚本 |
| 国学大师 | guoxuedashi.com | 古文字库 → 按字搜索 | 字形数量最多，包括异体字 |
| 书法字典 | sfds.cn | 搜字 → 选朝代 | 有历代书法家真迹字形 |
| 小篆字典 | 搜索"小篆字典 在线" | 按部首查 | 说文解字标准小篆 |

## 图片处理要求

从网上下载的字形图片通常不能直接用，需要统一处理。

### 尺寸与格式

- 输出格式：PNG（透明背景）或 JPG（白底）
- 尺寸：1024×1024 像素，正方形
- 字形居中，四周留白均匀（上下左右各留约 15% 空白）

### 处理步骤

```
1. 下载原始字形图片
2. 裁剪为正方形（如果原图是长方形）
3. 调整尺寸为 1024×1024
4. 如果背景复杂，抠图去背景，只留黑色字形
5. 字形居中，四周留白均匀
6. 保存为 PNG（透明背景）或 JPG（白底）
```

### 风格统一要求

- 同一个字的 4 张关键帧图，字形在画面中的位置、大小要一致
- 如果甲骨文是白底黑字，金文、小篆、楷书也都要白底黑字
- 不要一张白底一张黑底，否则 AI 过渡动画会闪烁

## 素材整理结构

```
素材库/
├── 取/
│   ├── 01_甲骨文.png    ← 商朝，手+耳的图画
│   ├── 02_金文.png      ← 西周，手+耳的铭文
│   ├── 03_小篆.png      ← 秦朝，圆润的篆书
│   └── 04_楷书.png      ← 现代，方正的楷书
├── 采/
│   ├── 01_甲骨文.png
│   ├── 02_金文.png
│   ├── 03_小篆.png
│   └── 04_楷书.png
└── 休/
    └── ...
```

命名规范：`{序号}_{朝代}.png`，序号保证排列顺序正确。

## 如果找不到某个阶段的字形

不是每个字都有完整的甲骨文→金文→小篆→楷书四个阶段：

- **没有甲骨文**：用金文作为起点，只做 3 段动画
- **没有金文**：跳过金文，甲骨文直接到小篆
- **字形差异太小**（如小篆和楷书几乎一样）：合并为一段，只做 2 段动画
- **原则**：有明显视觉变化的阶段才做动画，变化太小的阶段跳过
