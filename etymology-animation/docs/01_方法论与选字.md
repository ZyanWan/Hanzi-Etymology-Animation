# 方法论与选字

## 核心理念

汉字是世界上唯一仍在使用的象形文字系统。每个汉字的演变都是一部微型视觉故事。通过动画呈现这个故事，学生能直观理解：

- **这个字为什么长这样** — 字形来源
- **这个字为什么有这个意思** — 本义与引申义的关系
- **这个字怎么记** — 视觉锚定记忆

## 选字策略

### 优先级分层

| 层级 | 类型 | 特点 | 动画价值 | 示例 |
|------|------|------|----------|------|
| **P0** | 独体象形字 | 一个物体的简笔画 | ★★★ 最直观 | 日、月、山、水、火、人、大、木、目、耳 |
| **P1** | 会意字 | 两个象形部件组合 | ★★★ 故事性强 | 取、采、休、明、森、男、安、家、信、看 |
| **P2** | 指事字 | 象形字上加标记符号 | ★★ 有巧思 | 上、下、本、末、刃、寸 |
| **P3** | 形声字 | 一半表义一半表音 | ★ 动画价值低 | 江、河、湖、海 |

**建议**：先做 P0 和 P1，验证效果后再考虑 P2。P3 不推荐做。

## 每个字需要研究的内容

选好字后，逐字查资料并填写：

```
【字】：取
【读音】：qǔ
【本义】：割取敌人的耳朵（古代战争中割耳计功）
【字形来源】：
  - 甲骨文：左边一只耳朵，右边一只手
  - 金文：形状规整，耳和手更像符号
  - 小篆：圆润统一，"耳"在左"又"在右
  - 楷书：今天的"取"
【演变特点】：
  - 甲骨文→金文：最大变化，从图画变符号
  - 金文→小篆：线条变圆润
  - 小篆→楷书：圆笔变方笔
【故事脚本】：
  商朝士兵打完仗，一只手抓着敌人的耳朵，
  慢慢变成了今天我们写的"取"字。
```

## 查资料方法

1. 打开 [汉典 zdic.net](https://www.zdic.net)
2. 搜索目标字
3. 点击"字源"标签页
4. 记录甲骨文、金文、小篆的字形图片和说明文字
5. 交叉参考 [象形字典 vividict.com](http://www.vividict.com) 的字源故事
